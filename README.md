## **How to Run**
1. Clone the repository:  
   ```bash
   git clone https://github.com/khanboy1989/SwiftDataSeries.git
   cd SwiftDataSeries
   ```
2. Open the project in **Xcode**.
3. Run the app on an **iOS 16+ simulator or device**.
